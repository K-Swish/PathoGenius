About PathoGenius

PathoGenius is a cutting-edge platform dedicated to the study and analysis of the Filoviridae family, with a focus on Ebolavirus genomes. Researchers can explore our powerful genome analysis database and use our intuitive installer to set up local databases for in-depth study.

Genome Analysis Database

Our genome analysis database provides detailed insights into Ebolavirus genomes, including annotations, synteny maps, and sequence alignments.

Database Installer

Set up your local Ebolavirus database with ease using our installer. Follow the instructions to get started.
